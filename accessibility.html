Соответствие стандарту ТИ 5568 (AA)

Компания Kedem Tours придаёт большое значение равному доступу к своим услугам для всех пользователей, включая людей с ограниченными возможностями. Мы прилагаем значительные усилия, чтобы сделать сайт максимально удобным и доступным.

Сайт соответствует требованиям израильского стандарта № 5568 («Руководство по обеспечению доступности контента в интернете») и международным рекомендациям WCAG 2.1 уровня AA (Web Content Accessibility Guidelines).

Декларация последний раз обновлена: июнь 2025 г.

Меры по обеспечению доступности

На сайте реализованы следующие меры для обеспечения доступности:

  • Контрастность цветов соответствует требованиям WCAG 2.1 AA (соотношение не менее 4,5:1 для обычного текста)
  • Все изображения снабжены альтернативным описанием (атрибут alt)
  • Полноценная навигация с клавиатуры — доступны все интерактивные элементы
  • Ссылка «Перейти к содержимому» в начале каждой страницы
  • Семантическая HTML-разметка с ARIA-ориентирами (banner, main, contentinfo)
  • Все поля форм связаны с метками (label), указаны обязательные поля
  • Кнопки и ссылки имеют понятные текстовые описания
  • Ошибки и подтверждения в формах объявляются программами чтения с экрана (aria-live)
  • Поддержка системной настройки «уменьшить движение» — анимации отключаются автоматически
  • Язык страницы задан в коде (атрибут lang)
  • Текст можно увеличить до 200% без потери контента и функциональности
  • Диалоговые окна управляются с клавиатуры, фокус возвращается после закрытия

Виджет доступности

На каждой странице сайта в левом нижнем углу расположена кнопка ♿. Нажмите её, чтобы открыть меню настроек доступности:

  • Размер текста — увеличение (A+) или уменьшение (A−) шрифта
  • Высокий контраст — усиленный режим контрастности для слабовидящих
  • Остановить анимацию — отключение всех движущихся элементов
  • Выделить ссылки — подчёркивание и обводка всех ссылок на странице
Настройки сохраняются в памяти браузера и применяются при каждом посещении сайта.

Программы чтения с экрана

Сайт проверен на совместимость со следующими программами чтения с экрана:

  • NVDA (Windows) — бесплатная, рекомендуется с браузером Firefox или Chrome
  • JAWS (Windows) — профессиональная программа для незрячих пользователей
  • VoiceOver (iOS / macOS) — встроенная в устройства Apple
  • TalkBack (Android) — встроенная в устройства Android

Рекомендуемые браузеры: Google Chrome, Mozilla Firefox, Microsoft Edge.

Известные ограничения

Несмотря на прилагаемые усилия, на сайте могут существовать следующие ограничения:

  • Часть изображений, загружаемых динамически из внешних источников, может не иметь подробного альтернативного описания
  • Встроенные виджеты сторонних сервисов (отзывы Google, Facebook) находятся вне нашего контроля
  • Видеоматериалы в будущем будут снабжены субтитрами

Мы работаем над устранением имеющихся ограничений и регулярно обновляем сайт.
