===============================================
  KEDEM TOURS — Сборка Android приложений
===============================================

ПАПКИ:
  client/  — приложение для клиентов
  admin/   — кабинет администратора

===============================================
ШАГ 1 — Установи нужные программы (один раз)
===============================================

1. Node.js — https://nodejs.org
   Скачай версию LTS, установи с настройками по умолчанию.

2. Android Studio — https://developer.android.com/studio
   Установи с настройками по умолчанию.
   При первом запуске он сам скачает Android SDK (~1-2 ГБ).

===============================================
ШАГ 2 — Собери приложение для КЛИЕНТОВ
===============================================

1. Открой папку "client" в Проводнике
2. Дважды кликни на файл setup.bat
3. Подожди пока откроется Android Studio
4. В Android Studio выбери: Build → Build Bundle/APK → Build APK(s)
5. Подожди 2-5 минут
6. Нажми "locate" — откроется папка с файлом kedem-tours-client-debug.apk

===============================================
ШАГ 3 — Собери приложение КАБИНЕТ
===============================================

Повтори шаг 2, но в папке "admin"

===============================================
ШАГ 4 — Установи APK на телефон
===============================================

1. Отправь .apk файл себе (через Telegram, email и т.д.)
2. Открой на телефоне
3. Android спросит: "Разрешить установку из неизвестных источников" → Разрешить
4. Установить

===============================================
ВАЖНО
===============================================

- Оба приложения работают через интернет с сайта kedem-tours.com
- Без интернета не работают (это нормально для данного типа приложений)
- При обновлении сайта приложения обновятся автоматически
- APK можно пересобрать в любой момент
