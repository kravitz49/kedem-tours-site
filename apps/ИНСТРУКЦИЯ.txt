===============================================================
  KEDEM TOURS — Нативные Android приложения (React Native)
===============================================================

ПАПКИ:
  client-native/  — приложение для клиентов
  admin-native/   — кабинет администратора


===============================================================
  ЧТО ЕСТЬ В КАЖДОМ ПРИЛОЖЕНИИ
===============================================================

КЛИЕНТСКОЕ (client-native):
  • Список экскурсий с поиском
  • Экран экскурсии: фото, описание, форма записи (нативная)
  • Экран отзывов: список + форма с оценкой звёздами

КАБИНЕТ (admin-native):
  • Вход по паролю
  • Заказы: список, поиск, статистика, удаление
  • Экскурсии: добавить, редактировать, удалить
  • Отзывы: модерация (опубликовать/скрыть/удалить)
  • Настройки: логотип, фото шапки, остановки посадки


===============================================================
  ШАГ 1 — Установи Node.js (один раз)
===============================================================

  Скачай LTS версию: https://nodejs.org
  Установи с настройками по умолчанию.


===============================================================
  ШАГ 2 — Быстрый запуск на телефоне (без Android Studio!)
===============================================================

  1. Установи на телефон приложение "Expo Go" из Google Play
  2. Убедись что телефон и компьютер в одной Wi-Fi сети
  3. Открой папку client-native (или admin-native)
  4. Дважды кликни setup.bat
  5. Когда появится QR-код — сканируй его через Expo Go
  6. Приложение откроется на телефоне немедленно!

  Это НЕ финальный APK, но полностью рабочее приложение для
  тестирования. Все функции работают.


===============================================================
  ШАГ 3 — Сборка финального APK (устанавливается как обычное)
===============================================================

  ВАРИАНТ А — Через облако EAS (не нужен Android Studio):
  -------------------------------------------------------
  1. Создай бесплатный аккаунт на https://expo.dev
  2. В папке приложения запусти: eas login
  3. Запусти: eas build --platform android --profile preview
  4. Через 10-15 минут скачай APK по ссылке

  ВАРИАНТ Б — Через Android Studio (локально):
  --------------------------------------------
  1. Установи Android Studio: https://developer.android.com/studio
  2. В папке приложения запусти: npx expo run:android
  3. APK будет в папке android/app/build/outputs/apk/

  Рекомендую Вариант А — быстрее и проще.


===============================================================
  УСТАНОВКА APK НА ТЕЛЕФОН
===============================================================

  1. Отправь .apk файл себе (Telegram, email и т.д.)
  2. Открой на телефоне
  3. Разреши установку из неизвестных источников
  4. Установить → Открыть
